HTML Summary

HTML (HyperText Markup Language) was the first language created to build the structure of web pages. It was invented by Tim Berners-Lee in 1991 to make creating web pages easier without using a complex programming language. HTML is not a programming language, it is a Markup Language. Its main job is to create the structure of a web page. With HTML we can display text, images, videos, links, create forms, tables, lists, and organize the content using semantic tags.
